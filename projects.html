Land Use Inventory

The Land Use Inventory Program, developed in C, manages land use data by allowing users to add, search, and view lots, and calculates areas using the shoelace formula. It features a user-friendly menu with options to add, search, and view lots, ensuring accurate data by verifying land use categories and prompting users for inconsistencies. This system combines insights from previous studies with efficient spatial analysis, contributing to urban planning and sustainable development.

Numerical Methods - own preffered lanuage

My Numerical Methods Solver is a Python-based desktop app that helps users find equation roots using methods like Bisection, Newton-Raphson, Secant, and Graphical analysis. It allows users to input functions or data points, then visualizes each method step-by-step through dynamic plots and tables. This tool simplifies complex math, enhances learning, and supports efficient problem-solving for students and educators..

Numerical Methods - matlab

My Numerical Methods Root Finder is a MATLAB GUI application that helps students and educators solve nonlinear equations using classic root-finding methods. It supports user-defined functions, visual plotting, and step-by-step iteration outputs. By combining multiple methods into one interactive tool, it simplifies learning, enhances engagement, and promotes a deeper understanding of numerical analysis.

Sumobot and Hockeybot

The Sumobot / Hockeybot is a dual-mode Arduino robot that switches between autonomous sumo battles and Bluetooth-controlled hockey play. It combines sensor-based logic with wireless control, making it a fun and educational tool for learning robotics and embedded systems.
